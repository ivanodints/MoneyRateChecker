Этот проект представляет собой чекер курса валют.
Программа заходит на сайт ЦБ РФ и парсит xml с курсами валют.
При вводе в адресную строку интересующего нас "charCode" валюты мы получаем соответствующий результат.

Инструкция по запуску.
1. Запускаем CbrfApplication
2. В адресной строке любого браузера вбиваем адресс http://localhost:8010/cbrf/api/v1/rate/{***}
гда *** - charcode необходимой валюты.
